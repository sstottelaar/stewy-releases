# Stewy releases

Download the latest signed and notarized Stewy build for macOS:

## [Download the latest Stewy release](https://github.com/sstottelaar/stewy-releases/releases/latest)

The universal DMG supports both Apple Silicon and Intel Macs. Open the latest release page, download the `.dmg`, and drag Stewy to your Applications folder.

Current release: [Stewy v0.5.1](https://github.com/sstottelaar/stewy-releases/releases/tag/stewy-v0.5.1)

This repository contains Stewy installers and automatic update artifacts.
